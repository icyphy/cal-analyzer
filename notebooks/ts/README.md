# Running Jupyter Notebook in TypeScript in VSCode / Cursor

## Prerequisites

### 1. Install `Jupyter` from `ms-toolsai` in VSCode / Cursor extensions.

### 2. Install `tslab`
```bash
npm install -g tslab
```

## How to run

Open a TS notebook, and click top-right to select the `tslab` kernel. Then the notebook is ready to execute.
